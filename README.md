###  Hola!!Les presento esta aplicacion interactiva para buscar personajes de Harry Potter!!!

 Me agrada presentar el proyecto final del módulo 3 de Adalab de la promoción Salas!! Esta vez el reto fue realizar una aplicacion de busqueda de personajes de Harry Potter, aplicando todo lo aprendido de React durante el modulo.


## En el proyecto podrás encontrar:

* Uso de las etiquetas de html
* Maquetación con Sass
* Uso de JavaScript
* Uso de React: componentes, lifting, hooks y mas!
* Uso de git para para el control de versiones del proyecto y publicación del resultado en GitHub Pages.


**Pasos para arrancar el proyecto:**

Antes de arrancar el proyecto debes instalar dependencias de Node, **El proyecto hay que arrancarlo cada vez que te pongas a programar.** Para ello ejecuta el comando:

```bash
npm start
```
